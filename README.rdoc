== README

Ruby on Rails app that gets image and video data from an endpoint provided by the Instagram API.

Details:
* ruby version 2.2.2
* rails version 4.2.1

<b>gem dependencies:</b> 

  'gem rest-client'       #for get put post delete requests/interacting with API
  
  'gem pg'                #for database
  
  'gem rails_12factor'    #helps pg
